Design de Software é o curso onde é introduzida e desenvolvida a habilidade de programação em Python e seus módulos. O Projeto 3 de Design de Software foi o primeiro momento no qual se desenvolveu um programa envolvendo uma interface. Os alunos podiam escolher entre desenvolver um software, site ou jogo, podendo utilizar servidores web para isso. Além da linguagem Python, esse projeto ensinou os alunos a terem autonomia e conseguirem aprender a aprender, para conseguir realizar as tarefas específicas de seu projeto. Os grupos aprenderam a utilizar ferramentas como Tkinter, Pygame, Firebase, entre outros. A avaliação final foi feita por meio de um Demo Day, que requisitou apresentação oral e demonstração do funcionamento do projeto.
