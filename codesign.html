Co-Design de Aplicativos é um curso que ensina conceitos fundamentais de design e linguagens de programação para criar sites e aplicativos web. O projeto 1 tem como objetivo principal a criação de um site que apresenta os projetos trabalhados no insper, divulgando para o público as atividades acadêmicas trabalhadas pelos alunos do curso de engenharia. O aluno aprende habilidades importantes de programação nas linguagens Html, Css e JavaScript, criando sites com um caráter responsivo e com design fundamentado em teorias estudadas ao longo do semestre. O aluno além de criar interfaces visualmente agradáveis, aprende linguagens de programação importantes para várias aplicações junto com o grupo, estimulando a comunicação e o trabalho em grupo.
