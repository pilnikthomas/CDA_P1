Ciência dos Dados é um curso de estatística com o foco em manipulação e análise de dados usando ferramentas de programação. O projeto 1 tem como principal objetivo aprender a manipulação de dados usando a biblioteca Pandas do Python. O aluno importa uma base de dados extensa do IBGE, a PNAD e analisa o comportamento, ou do uso da tecnologia ou da segurança alimentar, com base nas variáveis escolhidas. Além da análise gráfica o aluno adquire habilidades importantes para o mercado de trabalho iniciando a sua jornada no mundo Big Data.
